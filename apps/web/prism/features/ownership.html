04 / 05 · Reuse & versions

Ownership is never ambiguous.

Prism shares everything the same way — components, design tokens, references. Mark it linkable. Place it as a live instance. Detach it to own it. Three verbs, everywhere, with guarantees.

The problem

Reuse, the haunted version

You duplicate a card and the copies drift apart. You reuse an instance and someone edits the master, silently changing ten screens. You delete a “copy” and the original vanishes with it. Tokens follow one sharing model, components another, assets a third.

Canvas tools make reuse a game of memory: which one of these is real?

The approach

Three verbs, one mechanism

A workspace component, a brand token, a reference image — anything marked linkable can be placed into a lower scope as a linked instance: a live window to its master, never a secret copy. Editing happens at the master's home, always; a back button round-trips you there and back. Detach converts an instance into an independent, fully-owned local copy.

You can always tell whether you're looking at a reference or the real thing — and where the real thing lives.
Componentization at work
Law · ownership and origin are never ambiguous

Versions that keep their promises

The same honesty runs through versioning. A version is always a copy of the frame — you choose how its components relate to the originals:

  • Linked version — inner components become live instances of the originals. Edit a master once, see it everywhere.
  • Copy version — fully independent, by law. Its components are re-created as its own; deleting one can never touch what it was copied from.
  • Promotion moves the crown — make any version the main and the components move with it. The new main truly owns its content; the old main becomes a version that references it.
  • No silent breakage — unlink or delete a shared item and Prism asks, per placement, whether to keep an owned copy there or remove it.

The result

Reuse without the haunting.

Change a master button and watch every linked screen follow. Detach one and know — with certainty — that it's yours alone. Nothing drifts, nothing breaks silently, nothing is secretly shared.
